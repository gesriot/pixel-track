# Pixel Track

Приложение на Python/PySide6 для ручного покадрового измерения движения объектов по серии изображений.

Текущий срез проекта после Спринта 2:
- открытие папки с кадрами;
- поддержка стандартных форматов изображений Qt;
- натуральная сортировка последовательности кадров;
- навигация по кадрам;
- масштабирование с сохранением зума между кадрами;
- настройка FPS из интерфейса;
- режим калибровки на текущем кадре;
- ввод длины калибровочного отрезка в метрах;
- отображение пиксельной длины и масштаба в правой панели.

## Запуск

Если окружение уже создано через `uv`, достаточно:

```bash
source .venv/bin/activate
python main.py
```

После синхронизации зависимостей через `uv sync` можно запускать и так:

```bash
uv run pixel-track
```
