# Pixel Track

Инструмент на Python/PySide6 для ручного покадрового измерения движения объектов по серии изображений.

## Возможности

- Открытие папки с кадрами; поддерживаются все форматы изображений Qt
- Натуральная сортировка последовательности кадров
- Навигация по кадрам, кеш соседних кадров для быстрой перелистки
- Масштабирование с сохранением позиции между кадрами
- Настройка FPS
- Калибровка масштаба на любом кадре с наследованием на последующие
- Ручная разметка точек `Previous` и `Current` для измерения шага движения
- Режим `Edit Handles` — перетаскивание точек калибровки и измерения прямо на кадре
- Расчёт расстояния и скорости (м/с)
- Журнал шагов и график скорости по времени
- Отмена и повтор действий (Undo/Redo)
- Сохранение и загрузка проекта в JSON; файл проекта переносимый — работает после перемещения вместе с папкой кадров
- Экспорт измерений в CSV

## Запуск

```bash
uv run pixel-track
```

Или через активированное окружение:

```bash
source .venv/bin/activate
python main.py
```

## Горячие клавиши

| Клавиша | Действие |
|---|---|
| `Left` / `Right` | Предыдущий / следующий кадр |
| `PageUp` / `PageDown` | −10 / +10 кадров |
| `Home` / `End` | Первый / последний кадр |
| `N` | Режим Navigate |
| `C` | Режим Calibrate |
| `P` | Режим Mark Previous |
| `M` | Режим Mark Current |
| `E` | Режим Edit Handles |
| `Ctrl+=` / `Ctrl+-` | Увеличить / уменьшить масштаб |
| `Ctrl+0` | Вписать кадр в окно |
| `Ctrl+Z` / `Ctrl+Y` | Undo / Redo |

## Меню File

- `Open Frames Folder…` — открыть папку с кадрами
- `Open Project…` — загрузить сохранённый проект
- `Save Project` / `Save Project As…` — сохранить проект
- `Export Measurements CSV…` — экспортировать измерения

## Структура

```
pixel_track/
  analysis.py        — расчёт метрик сегментов
  app.py             — точка входа
  controller.py      — бизнес-логика, QUndoStack
  frame_sequence.py  — сбор и сортировка кадров
  model.py           — модели данных
  project_io.py      — сериализация JSON и CSV
  undo_commands.py   — команды Undo/Redo
  ui/
    image_view.py    — виджет просмотра кадра с оверлеем
    main_window.py   — главное окно
    speed_plot.py    — график скорости
tests/
```
